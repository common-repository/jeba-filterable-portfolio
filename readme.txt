Jeba Cute Filter is an awesome Filter, super lightweight plugin for your wordpress website Filter portfolio Plugin. 

== Description ==

This plugin will enable awesome Filter in your wordpress theme. You can use the jeba cute Filter via shortcode in everywhere you want, even in theme files. 

Wanna see how it works? Click here: http://prowpexpert.com/jeba-filter-portfolio
 Jeba other wordpress plugin link: https://wordpress.org/plugins/jeba

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want use the jeba cute Filter in your theme php, Place `<?php echo do_shortcode('[jeba_filter]'); ?>` in your templates.
5. In the [jeba_filter] shortcode you can change post_type="" 
Shortcodes

<strong>Use Jeba Cute Filter shortcode</strong>
<pre>[jeba_filter post_type=""]</pre>

== Frequently Asked Questions ==
= How to use the plugin? =
After install & activation the plugin get a custom post menu in dashboard name "JebaFilter". Add here some your item from add new. Now use the shorstcode [jeba_filter] where you want to the filter item.
